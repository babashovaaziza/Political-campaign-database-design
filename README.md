# Political Campaign Database Design

## Overview
This project presents the design of a normalized relational database for managing political campaigns, elections, candidates, donations, voting activities, media advertisements, and public feedback.

## Features
- Election Management
- Candidate Management
- Political Parties
- Campaign Events
- Donations and Expenses
- Voting Records
- Media and Advertisements
- Public Feedback and Surveys

## Database Design
The database follows Third Normal Form (3NF) and uses:
- Primary Keys
- Foreign Keys
- Unique Constraints
- Check Constraints
